# Project Manager - CodeIgniter Backend

## Overview

**Project Manager** is a lightweight and efficient backend application built using the CodeIgniter framework. Designed for managing projects, tasks, and teams, it provides essential features for streamlined collaboration and productivity. This backend is highly adaptable and can be extended for integration with mobile applications built in React Native.

## Features

- **Project Management:** Create, edit, and manage multiple projects with ease.
- **Task Management:** Assign, update, and track tasks across teams.
- **User Management:** Handle user roles, permissions, and team assignments.
- **Real-time Insights:** Dashboard with project progress and team activity summaries.
- **API Ready:** RESTful APIs designed for seamless integration with React Native and other frontends.

## Why CodeIgniter?

CodeIgniter is known for its lightweight and high-performance nature, making it an excellent choice for backends requiring efficiency and speed. Its simplicity ensures rapid development, while its robust architecture supports scalability for larger applications.

## React Native Integration

This backend is built to support mobile application development, particularly with React Native, offering:

- **Comprehensive API Endpoints:** Simplified integration with mobile apps.
- **Authentication:** Ready-to-use authentication system with token-based security.
- **Scalability:** Easily adaptable for adding new features and handling more users.
- **Mobile Notifications:** Extendable for integrating push notifications and real-time updates.

## Technologies Used

- **Framework:** CodeIgniter
- **Database:** MySQL
- **Authentication:** Token-based (JWT-ready)
- **Mobile Integration:** Compatible with React Native for mobile app development
